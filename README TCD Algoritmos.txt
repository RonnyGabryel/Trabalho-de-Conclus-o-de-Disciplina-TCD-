COMO USAR O PROGRAMA
==============================================

1. Rode o programa
2. Escolha a opção 1 pra carregar um arquivo
3. Use as opções 2 e 3 pra buscar e ordenar
4. A opção 6 roda todos os testes de uma vez
5. A opção 4 gera um relatório automático

Formato do arquivo de dados:
Só colocar os números, um por linha:
42
17
89
5
